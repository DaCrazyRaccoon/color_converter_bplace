# WPlace Pixel Art Color Converter

Convert the colors of your images, pixel art or any creation to the available in Wplace.

## Funcionalidades

- Color conversion;
- Real time preview;
- Scaling;
- Dithering;
- Create or edit images (Studio);
- Store your images locally(Gallery);
